A fine-tuned Phi-3 AI Assistant built by Chandrakala, deployed using Hugging Face and Streamlit, capable of human-like, emotionally warm responses.

----------------------------------------------------------------------------------------------------------------------------------------------------

**Features**

Conversational AI capable of answering questions and providing assistance.
Fine-tuned on custom datasets for personalized responses.
Deployed on Hugging Face Hub for easy access.
Interactive Streamlit interface for end-to-end user experience.
Supports multi-turn conversations and maintains chat history.

----------------------------------------------------------------------------------------------------------------------------------------------------

**Technologies Used**

Python 3.x
Transformers (Hugging Face)
PyTorch
Streamlit
Docker (optional, for containerized deployment)

----------------------------------------------------------------------------------------------------------------------------------------------------

**Setup & Installation**

***1. Clone the repository***

git clone (https://github.com/chandra1024448/AI-Assistant/tree/main)

cd AI_Assistant

***2. Install dependencies***

pip install -r requirements.txt

***3. Run the app***

streamlit run streamlit_app.py

***Optional: Run with Docker***

docker build -t ai-assistant .
docker run -p 8501:8501 ai-assistant

----------------------------------------------------------------------------------------------------------------------------------------------------

**Usage**

Type your messages in the chat input.
Ask questions about AI, finance, or general queries.
The assistant responds accurately, concisely, and warmly.
